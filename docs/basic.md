# 設計

まず何を作りたいかというと 計算機

# 基本機能（必須）

1. **数字入力（0-9）**

   - 数字ボタンで数値を入力

2. **四則演算**

   - 足し算（+）
   - 引き算（-）
   - 掛け算（×）
   - 割り算（÷）

3. **等号（=）**

   - 計算結果を表示

4. **クリア機能**

   - `C`（Clear）：すべてリセット
   - `AC`（All Clear）：全クリア（履歴含む）

5. **小数点（.）**
   - 小数の入力

# 追加機能（あると便利）

6. **バックスペース/削除**

   - `⌫` または `DEL`：最後の入力を削除

7. **符号反転（±）**

   - 正負を切り替え

8. **パーセント（%）**

   - パーセント計算

9. **表示エリア**
   - 現在の入力値と計算結果を表示

# 実装のポイント

- **連続計算**：`1 + 2 = 3` の後、`+ 5 =` で `8` になる
- **演算子の変更**：`1 +` の後に `-` を押すと演算子が切り替わる
- **エラーハンドリング**：ゼロ除算などのエラー表示
- **入力の制限**：小数点の重複入力の防止
